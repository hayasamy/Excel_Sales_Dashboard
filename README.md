# Project Overview
This project presents a sales dashboard built in Microsoft Excel using Pivot Tables and Pivot Charts.  
The dashboard analyzes sales performance, customer purchasing behavior, product popularity, and delivery efficiency.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel formulas

# Business Questions Answered
- What is the sales trend over time?
- How do customers prefer to place orders?
- How much do customers buy?
- Which products are most popular?
- How long does delivery take on average?

# Dashboard Features
- Sales percentage by order mode
- Sales by product category
- Sales trend by date
- Interactive filters for order mode
- KPI summary section

# Key Insights
- App-based orders generated the highest share of sales
- Sneakers and Jeans were the top-selling products
- February recorded the highest sales volume
- Average delivery time remained under 3 days

# Project Purpose
This project demonstrates practical data analysis, KPI tracking, and dashboard storytelling using Excel.
